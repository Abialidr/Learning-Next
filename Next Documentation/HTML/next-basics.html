How to create Next App

use npx create-next-app to create new next app

what does create-next-app gives us

  1. package.json

    this is place where all the dependences of app is placed with other info too

  2. node modules

    all the dependences are installed here

  3. pages

    1. ./Api

      all the node/backend code is written here

    2. _app.js

      this is file which is entery point to whole project and dependences imported or files which are placed here will be reflected in all the components or pages of project

    3. _document.js

      This is the file that maintains the head section of all the projects, which means any CDN import is reflected in all of the projects.
      The difference between _app.js and _document.js is that _app.js is the entry point of the project's body side, whereas _document.js is the entry point of the project's head section.

    4. index.js

      first page which is loaded

  4. styles

    1. global.css

      styles written here will be applied to all the elemnts

      css modules are imported in this way :-
      import styles from "./[location(where the file is located)]"

      css modules are used in this way :-
      <componentName> className="styles.[className/whatever name of file is]" </componentName>

    2. home.module.css

      This module's base CSS file is only applied to the module in which it is imported.

  5. public

    all static resources like image, videos, mp3 etc are placed here

  6. .next

    This is the folder that is built when the project is ready for production. This file is deployed on the server. It can be built using "npm run build."

What is fast refresh provided in next js

Fast refresh is way that next js refreshes the output in dev environment.Fast refresh happens when ever you make new change and save it.whenever an error is occured then fast refresh will open a modal showing the exact error and where it has happened in your code

What is <Head> offered by next js

<Head> is basically head tag of html it works and act same as html head tag. title tag and link tag meta tags etc are supported in <Head> tag

<Head> is impored in this way :-
import <Head> from 'next/head
and used between <Head> </Head>
